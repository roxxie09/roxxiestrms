Request a stream or VOD

Request an event you'd like us to stream or upload as a VOD. Make sure to check the homepage & VODs page first before filing a request. To help us out, try to fill as much information as possible in the below form.

If you're not specific about what you want streamed, we won't stream it. Please request a specific event, i.e. Jake Paul vs. Mike Tyson instead of just "boxing".

For a successful request, read the following:

  1. Ensure the name or event title is specific and accurate: Instead of requesting just "football", you need to request a specific game or a set of games such as those in a tournament.
  2. Although not required, ensure the event channel & network is set: This helps us identify the best channel to use for the stream.
  3. Set the correct starting date & time: Identify the time for the event in your timezone, and put that in starting date & time. It will automatically convert to be correct for anyone reviewing requests.
  4. Ensure you give us enough time to put a stream on: Requesting at least 12 hours in advance is recommended for the highest likelihood of your stream being added to the website. Extremely late requests (i.e. minutes before, or during the event) will almost certainly not be added.
