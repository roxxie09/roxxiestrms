TEST PLAYER

The JWP Stream Tester supports testing & debugging secure HTTPS streams with JW Player only. HTTP streams may cause the Stream Tester to fail.

Widevine and PlayReady use DASH streams and FairPlay uses HLS streams. If both streams are provided the player will select the appropriate stream. If only one stream is provided, and the browser does not support the stream/DRM combination, the player will error. Learn more about platform compatibility here.
